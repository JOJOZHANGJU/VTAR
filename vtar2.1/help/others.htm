Wall and fluid property setup

     . Select Wall and fluid property setup  from the Configuration menu in the user interface.

     . The default properties of wall and fluid are saved in file /vtar/property.txt , user can modify this file by adding more types of fluid or modifying default properties.(Please pay attention to file format)
