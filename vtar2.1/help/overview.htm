Overview of VTAR graphical user interface

 

When you first open VTAR, it will display a window as the following illustration.

 

  The user interface of VTAR  has the following components:   

  •   Edit region for choice of sound category and area function data input

  •   Plot for calculated acoustic response based on area function setting and sound category choice

  •   Plot(s) for area function of vocal tract , the number of plots depends on the sound category

  •   Display area for formants information of acoustic response

  •   A marker measurements area  (not shown in the above figure, since the button for marker is not toggled)

  •   Buttons for some frequently used functions such as Exit, Save, Load, and Calculate

                Quit from VTAR
                     Load area function setting , and acoustic response and other configuration parameters, which are saved previously
                Save current area function setting , and acoustic response and other configuration parameters
                Calculate acoustic response

  •   A toolbar with buttons for convenient access to frequently used functions

                     Load area function setting , and acoustic response and other configuration parameters, which are saved previously
               Save current area function setting , and acoustic response and other configuration parameters
              Print and print preview
Zoom the data in plot  in and out
                Hold plots so that a comparison on acoustic response between different configurations can be made
                Clear those plots which are held
                Toggle the markers on and off
Set marker types (enabled by toggling on the button for marker)
                Calculate acoustic response
