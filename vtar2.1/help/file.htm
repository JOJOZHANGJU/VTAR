Input from file

The area function can also be loaded from file.  In the input file, users specify sound category, vocal tract geometry configuration, number of sections and corresponding area functions, and other related information.  Sample input files are given below:

  • Area function (.ARE) for one section (see /vtar/areafunction/template1.are)

  • Area function (.ARE) for one sound (see /vtar/areafunction/template2.are)

  • Vowel           (see example  /vtar/areafunction/vowel/EY.ARE)

  • Fricative        (see example  /vtar/areafunction/consonant/Example_1.ARE)

  • Nasal            (see example  /vtar/areafunction/nasal/M.ARE)

  • Rhotic /r/       (see example  /vtar/areafunction/_l/Example_1.ARE)

  • Lateral /l/       (see example  /vtar/areafunction/_r/Example_1.ARE)

VTAR also allows input of area function for arbitrary vocal tract geometry. Samples of arbitrary vocal tract area function input file are given below:

  • Arbitrary area function input file (/vtar/areafunction/template3.are )

  • Example of arbitrary area function input file for Vowel           (see   /vtar/areafunction/arbitrary/vowel.ARE)

  • Example of arbitrary area function input file for Nasal            (see   /vtar/areafunction/arbitrary/M.ARE)

  • Example of arbitrary area function input file for Rhotic /r/       (see  /vtar/areafunction/arbitrary/rhotic.ARE)

  • Example of arbitrary area function input file for Lateral /l/       (see  /vtar/areafunction/arbitrary/lateral.ARE)

 

 

Save and output

To save the results, click the 'save' button at the bottom left of the interface window, or go to menu 'file-save'. A dialog window will appear and save the results in a '.mat' file.  Specifically, the frequency and acoustic response are saved in VTsaved.f and VTsaved.AR, respectively.  The result is also saved in a text file with the same name.  A sample results file is given below:

  • Saved area function , acoustic response and other setting ( see /vtar/areafunction/savedexample1.mat and savedexample1.txt )

To load the results saved previously, click the 'load' button at the bottom left of the interface window, or go to menu 'file-load'.
