Introduction

VTAR is a Matlab-based computer program for vocal tract acoustic response calculation. Based on a frequency-domain vocal tract model [1], VTAR is able to model  various complex sounds such as nasals, rhotics and liquids.  With input in the form of vocal tract cross-sectional area functions, VTAR calculates the vocal tract acoustic response function, formant frequencies, and bandwidths.  The user-friendly interface allows flexible and directed data input for defined categories: vowels, consonant, nasals, nasalized sounds, rhotics, and  laterals. 

Several new features are included in VTAR 2.0, the latest version, which include: (1) acoustic sensitivity function calculation for formants, (2) area function modification for targeted formant pattern, (3) susceptance plot calculation, which is useful particularly for nasalized vowel analysis, (4) speech sound synthesis with source model options, and (5) addition of a new set of area function data for liquid sounds extracted from MR (Magnetic Resonance) images. These new features along with the user-friendly interface significantly enhance the usability of VTAR for both teaching and research purposes.
 

 VTAR has been tested in Matlab 6.5 on Windows.

 

 

 

Overview of VTAR interface window

 

Start and quit VTAR

    Start:

  • Start Matlab

  • Set VTAR directory '... /vtar' as current  directory or add it into search path of Matlab (addpath('.../vtar'))

  • In Matlab command window, type in 'vtar' and enter.

      Quit:

  • Press button   or select Exit VTAR from the File menu in the user interface

 

Vocal tract models and area function settings

 

Acoustic response calculation

  • To calculate acoustic response, press button  at toolbar  or button at the bottom of interface

 

Input from file

The area function can also be loaded from file.  In the input file, users specify sound category, vocal tract geometry configuration, number of sections and corresponding area functions, and other related information.  Sample input files are given below:

  • Area function (.ARE) for one section (see /vtar/areafunction/template1.are)

  • Area function (.ARE) for one sound (see /vtar/areafunction/template2.are)

  • Vowel           (see example  /vtar/areafunction/vowel/EY.ARE)

  • Fricative        (see example  /vtar/areafunction/consonant/Example_1.ARE)

  • Nasal            (see example  /vtar/areafunction/nasal/M.ARE)

  • Rhotic /r/       (see example  /vtar/areafunction/_l/Example_1.ARE)

  • Lateral /l/       (see example  /vtar/areafunction/_r/Example_1.ARE)

VTAR also allows input of area function for arbitrary vocal tract geometry. Samples of arbitrary vocal tract area function input file are given below:

  • Arbitrary area function input file (/vtar/areafunction/template3.are )

  • Example of arbitrary area function input file for Vowel           (see   /vtar/areafunction/arbitrary/vowel.ARE)

  • Example of arbitrary area function input file for Nasal            (see   /vtar/areafunction/arbitrary/M.ARE)

  • Example of arbitrary area function input file for Rhotic /r/       (see  /vtar/areafunction/arbitrary/rhotic.ARE)

  • Example of arbitrary area function input file for Lateral /l/       (see  /vtar/areafunction/arbitrary/lateral.ARE)

 

Save and output

To save the results, click the 'save' button at the bottom left of the interface window, or go to menu 'file-save'. A dialog window will appear and save the results in a '.mat' file.  Specifically, the frequency and acoustic response are saved in VTsaved.f and VTsaved.AR, respectively.  The result is also saved in a text file with the same name.  A sample results file is given below:
